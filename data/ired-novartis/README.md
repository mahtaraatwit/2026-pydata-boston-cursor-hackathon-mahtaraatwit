# IRED screening data (workshop demos)

CSV exports from an imine reductase (IRED) enzyme engineering campaign, included for **hands-on demos** in the Agentic Data Science workshop (e.g. EDA, structured workflows).

## Source and citation

Data originate from work performed at **Novartis** and are described in:

> Ma, E. J.; Siirola, E.; Moore, C.; et al. **Machine-Directed Evolution of an Imine Reductase for Activity and Stereoselectivity.** *ACS Catal.* **2021**, *11* (20), 12433–12445.
> DOI: [10.1021/acscatal.1c02786](https://pubs.acs.org/doi/abs/10.1021/acscatal.1c02786)

Use this repository copy only in line with the workshop context; for publication or redistribution of the raw tables, cite the paper and respect any applicable terms from the original study.

## Files

| File | Description |
|------|-------------|
| `ired-master-table.csv` | Row-level variant records (sample id, layout, plate position, mutation, protein sequence, plate code). |
| `layouts.csv` | Layout metadata (`layout_id`, `layout_code`, experiment type, plate size, comments). |
